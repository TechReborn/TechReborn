###### Nickel Ingot

import Mcitem from '/src/mcitem.mdx'
import Recipe from '/src/recipe.mdx'

- Nickel Ingot** used to craft several recipes. Tech Reborn
        doesn\'t have Nickel ore. It could be obtained either by
        processing several ores in
        <Mcitem id="techreborn:industrial_grinder"/>
        or by processing
        <Mcitem id="techreborn:copper_dust"/>,
        <Mcitem id="techreborn:gold_dust"/> or
        <Mcitem id="techreborn:iron_dust"/> in
        <Mcitem id="techreborn:industrial_centrifuge"/>
        if you do not have
        <Mcitem id="techreborn:industrial_grinder"/>
        yet.
