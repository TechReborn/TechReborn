###### Flint dust

- Flint dust** is a dust used to get
        <Mcitem id="techreborn:silicon_cell"/>.

\\\\

##### Recipe

<Mcitem id="minecraft:crafting_table"/> recipe: \\\\
`<recipe>` input techreborn:small_pile_of_flint_dust
techreborn:small_pile_of_flint_dust air input
techreborn:small_pile_of_flint_dust techreborn:small_pile_of_flint_dust
air input air air air output techreborn:flint_dust `</recipe>`

recipe: \\\\

        <Mcitem id="techreborn:industrial_electrolyzer"/>.
