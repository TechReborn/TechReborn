###### Uvarovite Dust

- Uvarovite Dust** is an intermediate product received by
        processing
        <Mcitem id="techreborn:yellow_garnet_dust"/>
        in
        <Mcitem id="techreborn:industrial_centrifuge"/>.

\\\\

##### Recipe

<Mcitem id="minecraft:crafting_table"/> recipe: \\\\
`<recipe>` input techreborn:small_pile_of_uvarovite_dust
techreborn:small_pile_of_uvarovite_dust air input
techreborn:small_pile_of_uvarovite_dust
techreborn:small_pile_of_uvarovite_dust air input air air air output
techreborn:uvarovite_dust `</recipe>` \\\\
<Mcitem id="techreborn:industrial_centrifuge"/>
recipe: \\\\
16x<Mcitem id="techreborn:yellow_garnet_dust"/> -\>
5x<Mcitem id="techreborn:andradite_dust"/> +
8x<Mcitem id="techreborn:grossular_dust"/> +
3x<Mcitem id="techreborn:uvarovite_dust"/> \\\\

        <Mcitem id="techreborn:industrial_electrolyzer"/>.\\\\
