###### Almandine Dust

- Almandine Dust** is an intermediate product received by
        processing
        <Mcitem id="techreborn:red_garnet_dust"/> in
        <Mcitem id="techreborn:industrial_centrifuge"/>.

\\\\

##### Recipe

<Mcitem id="minecraft:crafting_table"/> recipe: \\\\
`<recipe>` input techreborn:small_pile_of_almandine_dust
techreborn:small_pile_of_almandine_dust air input
techreborn:small_pile_of_almandine_dust
techreborn:small_pile_of_almandine_dust air input air air air output
techreborn:almandine_dust `</recipe>` \\\\
<Mcitem id="techreborn:industrial_centrifuge"/>
recipe: \\\\
16x<Mcitem id="techreborn:red_garnet_dust"/> -\>
3x<Mcitem id="techreborn:pyrope_dust"/> +
5x<Mcitem id="techreborn:almandine_dust"/> +
8x<Mcitem id="techreborn:spessartine_dust"/>

\\\\

##### Usage

- Almandine Dust** could be further processed in
        <Mcitem id="techreborn:industrial_electrolyzer"/>.

\\\\ 20x<Mcitem id="techreborn:almandine_dust"/> +
9x<Mcitem id="techreborn:empty_cell"/> -\>
3x<Mcitem id="techreborn:iron_dust"/> +
2x<Mcitem id="techreborn:aluminium_dust"/> +
3x<Mcitem id="techreborn:silicon_cell"/> +
6x<Mcitem id="techreborn:compressed_air_cell"/> \\\\

using <Mcitem id="minecraft:crafting_table"/>:
