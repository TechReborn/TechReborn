###### UU-Matter

import Mcitem from '/src/mcitem.mdx'
import Recipe from '/src/recipe.mdx'

- UU-Matter** is an item added by Tech Reborn. It is produced
        when the
        <Mcitem id="techreborn:matter_fabricator"/>
        is provided with adequate power and
        <Mcitem id="techreborn:scrap_box"/> or
        optionally <Mcitem id="techreborn:scrap"/>.

UU-Matter can be used to craft a variety of materials, or can be put in
a <Mcitem id="techreborn:fluid_replicator"/> to
replicate fluids.
