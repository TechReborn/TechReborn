##### Pyrite Ore

- Pyrite ore** is a common ore found in The Nether on layers
        10-126. It will drop
        <Mcitem id="techreborn:pyrite_dust"/>
        affected by Fortune enchant if mined normally. And will drop
        block itself if mined with SilkTouch enchant.
