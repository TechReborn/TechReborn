![Extractor](/mods/techreborn/extractor.png)

The **Extractor** is a machine that is used to process natural resources into another useful product.
For example, `![Sap](/mods/techreborn/sap.png)` **[Sap](items:part:sap "wikilink")** into `![Rubber](/mods/techreborn/rubber.png)` **[Rubber](items:part:rubber "wikilink")**. The **Extractor**
requires a power input of 32 E/tick to work and has internal buffer of 1k E.

The extractor can also be used to discard the contents of a
<Mcitem id="techreborn:cell"/> so that you can use
it again.
